# xo
 
Tic Tac Toe Game is a modern mobile implementation of the classic X-O game.
The application offers multiple gameplay modes including single-player with different difficulty levels and multiplayer mode.
The game is designed with a clean UI, smooth animations, and engaging sound effects to provide an enjoyable experience for players.
2️⃣ Game Modes

🔹 Easy Mode
In this mode the computer plays randomly, which makes the game simple and fun for beginners.
Features:
Random AI moves
Suitable for casual players
Allows players to easily win and learn the game

🔹 Hard Mode (AI Mode)
This mode uses the Minimax Algorithm, which is a well-known algorithm in game theory.
Features:
AI analyzes all possible moves
Always chooses the best move
Practically impossible to defeat
This mode provides a challenging experience for advanced players.

🔹 Multiplayer Mode
Players can compete against each other on the same device.
Features:
Player X vs Player O
Turn-based gameplay
Real-time interaction
3️⃣ Game Features
✨ Smooth Animations

The game includes smooth animations to improve user experience.
🎵 Sound Effects
Sound effects are used to enhance player interaction and feedback.
🎬 Lottie Animations
Lottie animations are integrated to create modern and attractive visual feedback.
🎨 Clean UI Design
The interface is simple and user-friendly.
🔄 Reset Game Option
Players can restart the game anytime.
